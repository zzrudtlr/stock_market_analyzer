추가 단계: AI 기반 테마(Theme) 분석 시스템 구현

기존 프로젝트 구조를 유지하면서
AI 기반 테마 강세/약세 분석 기능을 구현해줘.

목표:

시장 데이터를 기반으로
현재 강한 테마와 약한 테마를 자동 분석하고,
관련 종목을 그룹화하여
참고용 시장 흐름 분석 시스템을 구축한다.

중요:
- 투자 추천 시스템이 아니다
- “매수 추천”, “급등 확정”, “수익 보장” 표현 금지
- 참고용 흐름 분석만 제공
- 기존 DB 구조 유지
- DROP TABLE 금지
- ALTER TABLE 금지
- 기존 컬럼명 변경 금지
- 기존 파일 구조 유지

---------------------------------------------------------------

1. 테마 분석 서비스 생성

파일:
- services/theme_analysis_service.py

기능:
- 테마별 종목 그룹 분석
- 강세/약세 테마 탐지
- 거래량 기반 테마 탐지

- 시장 순환매 흐름 분석

3. 테마 관련 종목 매핑

방법:

1순위:
- theme_stocks 테이블 사용 가능 시 사용

컬럼 예시:
- theme_name
- stock_code
- stock_name

2순위:
- JSON 파일 기반 관리

파일:
- data/theme_mapping.json

예시:

{
  "AI": ["000660", "035420"],
  "반도체": ["005930", "000660"],
  "2차전지": ["373220", "066970"]
}

4. 테마 분석 기준

각 테마별 계산:

- 평균 daily_return
- 평균 return_5d
- 평균 return_20d
- 평균 거래량 증가율
- 상승 종목 수
- 하락 종목 수
- 강세 종목 비율
- 약세 종목 비율
- 평균 momentum_score
- 평균 trend_score
- 평균 risk_score

---------------------------------------------------------------

5. 강세 테마 판단 기준

예시:

- 평균 return_5d > 0
- 거래량 증가율 상승
- 강세 종목 비율 증가
- momentum_score 평균 양호
- trend_score 평균 양호

결과값:

- 매우 강세
- 강세 흐름
- 순환매 관심
- 혼조
- 약세 흐름
- 하락 주의

- trend_score 평균

7. AI 출력 내용

생성 항목:

- ai_theme_summary
- ai_theme_risk
- ai_theme_flow
- ai_theme_volume_comment
- ai_theme_rotation_comment

예시:

ai_theme_summary:
"최근 거래량 증가와 함께 AI 관련 종목 중심으로 단기 강세 흐름이 관찰됩니다."

ai_theme_risk:
"일부 종목 변동성이 확대되고 있어 단기 등락 가능성 점검이 필요합니다."

ai_theme_rotation_comment:
"시장 내 순환매 흐름이 일부 유입되는 모습이 관찰됩니다."
