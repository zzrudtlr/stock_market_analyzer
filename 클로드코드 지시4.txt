추가 단계: 초보자 친화형 투자 분석 UX 및 AI 설명 시스템 구현

기존 프로젝트 구조를 유지하면서
주식 초보자도 쉽게 이해할 수 있는
초보자 친화형 분석 UX 및 AI 설명 시스템을 구현해줘.

목표:
단순 수치와 지표를 보여주는 것을 넘어서
“왜 이런 분석 결과가 나왔는지”
초보자도 이해할 수 있도록 설명하는 시스템 구축.

중요:
- 투자 추천 시스템이 아니다
- “매수 추천”, “급등 확정”, “수익 보장”, “지금 사야함” 표현 금지
- 참고용 설명만 제공
- 기존 DB 구조 유지
- DROP TABLE 금지
- ALTER TABLE 금지
- 컬럼명 변경 금지
- 기존 파일 구조 유지

---------------------------------------------------------------

1단계: 용어 자동 설명 시스템 구현

파일:
- services/term_dictionary_service.py

목표:
주식 초보자를 위한 용어 설명 기능 구현.

설명 대상 예시:
- RSI
- MA5
- MA20
- MA60
- 거래량 증가율
- 외국인 순매수
- 기관 순매수
- PER
- PBR
- ROE
- 변동성
- 골든크로스
- 데드크로스
- 거래대금
- 공매도
- 상대강도

기능:
- 용어명 기반 설명 조회
- 초보자 친화형 설명 제공
- 짧은 설명 + 상세 설명 제공

예시:

RSI:
“최근 상승/하락 강도를 나타내는 지표입니다.
일반적으로 70 이상이면 단기 과열 가능성을 참고합니다.”

MA20:
“최근 20일 평균 가격 흐름입니다.
중기 추세를 확인할 때 많이 사용됩니다.”

API:
- GET /terms
- GET /terms/{term_name}

---------------------------------------------------------------

2단계: 초보자용 AI 설명 변환 기능 구현

파일:
- services/beginner_ai_explainer_service.py

목표:
복잡한 분석 데이터를
초보자가 이해하기 쉬운 자연어로 변환.

입력 데이터:
- RSI
- 거래량
- 이동평균선
- 수급
- 뉴스
- 위험도
- 테마 흐름
- 차트 패턴

AI 출력:
- beginner_summary
- beginner_positive_comment
- beginner_risk_comment
- beginner_market_comment

예시:

기존:
- MA20 돌파
- 거래량 증가
- RSI 상승

초보자 설명:
“최근 거래량이 평소보다 증가했고,
단기 흐름이 중기 흐름보다 강해지는 모습입니다.”

---------------------------------------------------------------

3단계: 긍정 요소 / 위험 요소 분리 표시 기능 구현

파일:
- services/pros_cons_analysis_service.py

목표:
초보자가 좋은 점과 위험 요소를 동시에 이해할 수 있도록 구성.

출력 구조:

positive_points:
- 거래량 증가
- 외국인 순매수
- 강한 테마 흐름

risk_points:
- 단기 급등 상태
- 변동성 확대
- 뉴스 과열 가능성

AI 문구 예시:

긍정 요소:
“거래량 증가와 함께 수급 흐름이 강화되고 있습니다.”

위험 요소:
“단기 가격 변동성이 확대되는 구간입니다.”

4단계: 투자 스타일 분류 기능 구현

파일:
- services/investment_style_analysis_service.py

목표:
종목의 성향을 초보자가 쉽게 이해하도록 분류.

분류 예시:

- 단기 변동형
- 추세 안정형
- 실적 성장형
- 테마 순환형
- 고위험 변동형
- 저변동 안정형

분석 기준:
- 변동성
- 거래량
- 실적
- 테마 강도
- 수급
- 이동평균 흐름

AI 문구 예시:
“변동성이 큰 편으로 단기 등락 가능성이 높은 유형입니다.”

7단계: 과거 유사 패턴 비교 기능 구현

- RSI 과열
- 테마 강세
- 외국인 수급

출력:

- past_uptrend_ratio
- past_sideways_ratio
- past_downtrend_ratio

예시:
“과거 유사 패턴에서는 상승 흐름 지속 비율이 상대적으로 높았습니다.”

8단계: 종목 한줄 요약 기능 구현

파일:

목표:
종목을 한 문장으로 설명.

예시:

에코프로:

9단계: AI 질문 기능 구현

파일:
- services/ai_question_answer_service.py

목표:
사용자가 직접 질문하면
현재 분석 데이터를 기반으로 답변 제공.

예시 질문:
- “왜 위험한가요?”
- “왜 강세인가요?”
- “외국인 순매수가 뭔가요?”
- “왜 거래량이 중요한가요?”
- “지금 과열 상태인가요?”

중요:
- 현재 분석 데이터 기반으로만 답변
- 투자 추천 금지

API:
- POST /ai/question
