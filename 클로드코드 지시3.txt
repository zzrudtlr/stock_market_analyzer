추가 단계: 투자 판단 보조용 고급 시장 분석 기능 구현

기존 프로젝트 구조를 유지하면서
투자 판단에 참고할 수 있는 고급 시장 분석 기능을 추가해줘.

중요:
- 투자 추천 시스템이 아니다
- “매수 추천”, “급등 확정”, “수익 보장”, “지금 사야함” 표현 금지
- 참고용 분석만 제공
- 기존 DB 구조 유지
- DROP TABLE 금지
- ALTER TABLE 금지
- 컬럼명 변경 금지
- 기존 파일 구조 유지

목표:
단순 가격 분석을 넘어
“왜 움직이는가”
를 설명할 수 있는 종합 시장 분석 시스템 구축.

분석 요소:
- 수급
- 뉴스
- 공시
- 실적
- 거래량
- 차트 패턴
- 테마 순환
- 시장 주도주

---------------------------------------------------------------

1단계: 수급 분석 기능 구현

파일:
- services/supply_demand_analysis_service.py

목표:
기관/외국인/개인 수급 흐름 분석.

분석 데이터:
- 외국인 순매수
- 기관 순매수
- 개인 순매수
- 연속 순매수 일수
- 프로그램 매매
- 공매도 증가율

계산 항목:
- foreign_net_buy
- institution_net_buy
- individual_net_buy
- foreign_buy_streak
- institution_buy_streak
- short_sell_ratio
- program_trade_ratio

AI 분석 문구 예시:
- “외국인 수급 유입이 지속되고 있습니다.”
- “기관 중심 매수 흐름이 관찰됩니다.”
- “개인 비중 증가로 변동성 확대 가능성이 존재합니다.”

결과 저장:
- supply_demand_analysis_results

upsert 기준:
- stock_code + analysis_date

2단계: 뉴스 감성 분석 기능 구현

파일:

목표:

분석 항목:
- positive_news_count
- negative_news_count
- neutral_news_count
- news_sentiment_score

AI 감성 분류:
- 호재
- 악재
- 중립

AI 문구 예시:
- “AI 관련 정책 뉴스 증가 흐름이 관찰됩니다.”
- “실적 우려 기사 비중이 증가하고 있습니다.”
- “수주 관련 뉴스 집중 발생이 확인됩니다.”

주의:
- 투자 추천 표현 금지

4단계: 차트 패턴 분석 기능 구현

탐지 패턴:

- MA60 돌파
- 골든크로스
- 데드크로스
- 눌림목 패턴

계산 항목:
- breakout_signal
- golden_cross_signal
- dead_cross_signal
- volume_breakout_signal
- new_high_signal

AI 문구 예시:

- “거래량 동반 상승 흐름이 나타나고 있습니다.”
- “중장기 이동평균선 정배열 흐름이 유지되고 있습니다.”

5단계: 시장 주도주 분석 기능 구현

파일:
- services/market_leader_analysis_service.py

목표:
실제 시장을 주도하는 종목 탐지.

분석 기준:
- 거래대금 증가율
- 시가총액 영향력
- 상대강도
- 거래량
- 테마 영향력

계산 항목:
- market_leader_score
- market_influence_score
- trading_value_rank
- theme_influence_score

결과:
- 오늘의 시장 주도주
- 시장 영향력 상위 종목
- 거래대금 상위 종목

AI 문구 예시:
- “반도체 대형주 중심의 시장 주도 흐름이 관찰됩니다.”
- “거래대금이 특정 테마에 집중되는 모습입니다.”

계산 항목:

- theme_flow_strength
- intraday_theme_rank

AI 문구 예시:

8단계: 종합 AI 시장 해설 기능 구현

파일:
- services/advanced_market_ai_service.py

목표:
모든 데이터를 종합하여
사람이 이해하기 쉬운 종합 시장 해설 생성.

종합 분석 요소:
- 수급
- 뉴스
- 공시
- 실적
- 거래량
- 차트 패턴
- 테마
- 시장 주도주

AI 출력:
- ai_market_flow_summary
- ai_stock_summary
- ai_risk_summary
- ai_theme_summary
- ai_supply_demand_summary

예시:
“반도체 테마 내 외국인 수급 유입과 AI 서버 관련 뉴스 증가가 동반되고 있으며, 거래량 증가와 함께 MA20 돌파 흐름이 관찰됩니다.”
